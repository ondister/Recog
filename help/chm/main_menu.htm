Главное меню программы

Главное меню программы располагается вверху экрана по всей его ширине. Оно состоит из кнопок и выпадающих списков:

Мастер потокового сканирования,

Работа с тестами,

Результаты,

В Excel,

Печать,

Справка и настройка,

Изменение режима меню,

Выход.

Главное меню недоступно во время прохождения теста на компьютере.

Кнопка «Результаты» открывает окно с результатами тестов.

Кнопка «Работа с тестами» открывает окно выбора тестов.

Кнопка «Мастер потокового сканирования» позволяет запустить мастер сканирования бланков одной методики.

Назначение остальных элементов описано в соответствующих разделах.
