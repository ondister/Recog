Окно выбора респондента

 При запуске теста в любом режиме перед началом ввода данных необходимо выбрать или ввести нового респондента в базу.

Если респондент уже присутствует в базе, его можно найти по начальным символам фамилии. Затем выберите его в списке левой кнопкой мыши и нажмите кнопку «Продолжить». Если необходимости в дальнейшем тестировании нет, нажмите кнопку «Отмена».

Данные респондента можно изменить. Для этого измените данные в полях и нажмите кнопку «Изменить».

Для добавления нового респондента нажмите кнопку «Добавить».

Не заполнять можно только поля «Имя» и «Отчество». Все остальные поля должны быть заполнены.

Если необходимого подразделения нет в списке, нажмите на кнопку  и введите необходимое подразделение, затем выберите его из списка подразделений.

Переход между полями можно осуществлять по нажатию клавиши «Enter», а переход между днем, месяцем и годом в поле «Дата рождения» можно осуществлять стрелками «Влево-Вправо» на клавиатуре.

После заполнения всех полей, нажмите на кнопку «Добавить».

Так же можно удалить данные респондента со всеми результатами тестов. Примите к сведению, что отменить эту операцию невозможно.
